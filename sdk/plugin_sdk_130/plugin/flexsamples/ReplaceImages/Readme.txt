Replace Images sample
ReadMe File
______________________________________________________________________

The Replace Images is an Adobe AIR application that can be used to 
to replace the images in an IDML file. You must build and export 
the application before running it.

***********************************************************
Building with Flash Builder 4.7 or later
***********************************************************

To import the project into Flash Builder:

  1. Start Flash Builder (or Eclipse if you have Flash Builder installed
     as a plug-in for Eclipse).
  2. Choose File > Import > Existing Projects into Workspace.
  3. Select 'Select root directory' and Browse to the <SDK>/flexsamples.
  4. Check the desired project.
  5. Click finish.

To export an installable AIR bundle from Flash Builder:

  1. Choose File > Export.
  2. Select Flash Builder > Release Build.
  3. Follow the wizard to completion to export a signed AIR package.

***********************************************************
Using the Applicatoin
***********************************************************

This application allows you to replace the images in an IDML file through drag 
and drop or by clicking on the "Replace Image" button and browsing to a new image.  
After replacing the image, the changes can be saved to a new IDML file. The 
examplefiles folder contains a sample IDML file and sample images to demonstrate 
how it works.
