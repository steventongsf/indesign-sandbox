Detailed Description

 Command to place multiple InDesign doc pages on one InDesign page.
Parameters
IID_IUIDDATAis the parent to place the InDesign doc pages. Parent should be a page, a spread, or any page item that can contain another item.
IID_IPOINTLISTDATAis the upper left location to place the pages. Only the first point is used.
IID_IPAGEINFODATAcontains the pages to place.
IID_IIMPORTRESOURCECMDDATAis the InDesign file used.
