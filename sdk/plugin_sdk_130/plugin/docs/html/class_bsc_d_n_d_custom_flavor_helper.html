virtual

performs the actual drag. Because we know that our custom flavor is masquerading and is really a page item we must take a copy of the page item, then move this copy to the drop zone. We know we have been dropped on a widget (because we are called as part of that widget responding to a drop of the custom flavor).

Parameters
targetIN the target for this drop.
controllerIN the drag drop controller that is mediating the drag.
