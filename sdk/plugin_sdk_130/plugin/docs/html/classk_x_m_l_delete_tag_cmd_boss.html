Detailed Description

 This command deletes XML tags, and removes them from the tag list; use

IXMLTagCommands::DeleteTag rather than processing this command. Also scans the structure tree and replace all instances of the elements with the specified replacement tag.

Item list output
Item-list contains UIDs of objects that were deleted
Observer notification

  • theSubject: Workspace (IWorkspace) that aggregates IXMLTagList, such as kDocWorkspaceBoss or kWorkspaceBoss
