Detailed Description

 Represents a spread-layer within a spread (kSpreadBoss), responsible for storing

page content items (for example, kSplineItemBoss, kGroupItemBoss), guides (kGuideItemBoss) and pages (kPageBoss). Spread-layers are owned by spreads (kSpreadBoss). The key interface aggregated on this class is ISpreadLayer.
