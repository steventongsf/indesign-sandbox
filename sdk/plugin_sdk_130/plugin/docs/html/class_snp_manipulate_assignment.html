Detailed Description

This snippet provides:

  • Create an assignemnt
  • Add new content to existing assignment
  • A snippet view of contents of an assignment
  • delete assignment. 
    <h2>Exercising</h2>

New Assignment

  1. Open an existing document or Create a new document and save
  2. Run the Manipulate Assignment snippet
  3. Choose the option New Assignment.
  4. Choose a file name the assignment to save to in the save as dialog
  5. Choose various assignment options
  6. If all is well, there should be no asserts (if in debug build).
  7. You should see a new assignment created in Assignment palette.
  8. Snippet runner log reports an assignment is created.

    Add to Assignment

  1. Open an existing document with assignment or continue from New Assignment above
  2. Select an image or story from the document window
  3. Run the Manipulate Assignment snippet
  4. Choose the option Add to Assignment
  5. Choose an assignment you want to add to
  6. There is an assert about fTransactionLevel is not 0 as of debug build 211, ingore it.
  7. You should see snippet runner log reports successful
  8. You should see a new assigned story is added to the assignment you choose in Assignment palette.
  9. Known issue: If your selection is not an image and story, you still can run, but nothing is added to the assignment.

    Inspect Assignment

  1. Open an existing document with assignment or continue from New Assignment above
  2. Add images or stories to assignment from Assignment palette or run above add to assingment snippet.
  3. Run the Manipulate Assignment snippet
  4. Choose the option Inspect Assignment
  5. Choose an assignment you want to inspect
  6. If all is well, there should be no asserts (if in debug build).
  7. Assignment information will be dumped to snippet runner log. Examine the result

    Delete Assignment

  1. Open an existing document with assignment or continue from New Assignment above
  2. Run the Manipulate Assignment snippet
  3. Choose the option Delete Assignment
  4. Choose an assignment you want to delete
  5. If all is well, there should be no asserts (if in debug build).
  6. You should see the assignment disappeared from Assignment palette.
  7. Snippet runner log will report assignment deleted.

    Running without InCopy plugins

If running in InDesign without AssignmentUI, InCopyBridge, InCopyBridgeUI, InCopyExport, InCopyExportUI plugin (these are normally installed under ...\plugins\incopyworkflow), the following behavior change is expected:

  1. Inspect Assignment, Delete assignment works as normal
  2. Add to assignment will fail becuase we can not export story without InCopyBridge. No item is added to assignment, but the SnippetRunner log will still report success (known issue) Manipulate assignment snippet. Create, inspect, delete an assignment, add the selected item (story/graphic) to assignment.
