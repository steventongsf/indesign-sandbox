Detailed Description

 This command deletes the content items associated with an element (IIDXMLElement),

but leaves the elements in place; use IXMLElementCommands::DeleteElementContent instead of processing this low-level command.

Item list output
Not used
Observer notification
Indirect, depends on the content item being deleted.
