63 // This interface has been combined with IID_ICOLORPICKERPANELMODKEYSTATEDATA
64 // Changed for Sherpa to implement shift-click
65 
66 // This data interface stores a mod key state and also keeps track for the global swatch
67 // associated with color definition changes due to shift-click
68 // Key state for the ColorPicker panel is stored as an int32.
69 // We save modifier key state only if they are relevant to the panel's observers.
70 // Starting with the lowest bit position:
71 // bit 0 = 1 if shift key pressed when mouse down
72 // bit 1 = 1 if command key pressed when mouse down
73 // bit 2 = 1 if alt/option key pressed when mouse down
74 // The data includes:
75 // boolean to indicate whether it starts with a global swatch
76 // UID of the global swatch
77 // Tint% of the global swatch if swatch was a base color
