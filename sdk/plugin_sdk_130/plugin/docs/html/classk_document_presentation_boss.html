IPMUnknown kDocumentPresentationBoss::IID_ISCRIPTOBSERVER 
     Windows are special beasts. The way we close documents is by closing all of the

windows on it. Unfortunately, this means that if we have refs on a window object, and we wait to get notified that the document is closing before we try to get the windows from the window list, we are told the document has no windows because they have been closed. At that point it is too late for us to release the refs on the window, so we need to have an observer.
