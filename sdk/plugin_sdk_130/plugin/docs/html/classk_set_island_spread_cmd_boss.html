Detailed Description

 This command edits the flag that indicates whether a spread

is an island spread or a normal spread. An island-spread contains a set of pages that are kept together in the island-spread as you add or remove pages that precede it. The pages contained in a normal spread will change as you add or remove pages that precede it.

Postcondition
The island-spread flag stored in the IBoolData/IID_IISLANDSPREAD interface on the affected spreads is edited.
Item list input
References the spreads (kSpreadBoss) to be edited.
