Detailed Description

 This command duplicates a page's contents by copying from a source

page (kPageBoss) to a destination spread (kSpreadBoss) or master spread (kMasterPagesBoss).

Postcondition
Page items are copied from the source page to the destination spread, objects on which the copied objects depend get copied too.
Item list input
None(IDuplicatePageCmdData identifies the object to be edited)
Item list output
None
Observer notification
  • theChange: kDuplicateSpreadCmdBoss (yes, currently kDuplicateSpreadCmdBoss is used to inform observers)
  • theSubject: destination spread (kSpreadBoss or kMasterPagesBoss) affected
  • protocol: IID_ISPREADLIST
  • changedBy: command instance
Responder notification
  • kDuplicateSpreadSignalResponderService
