Detailed Description

 This boss class provides the actions invoked by this plug-in's

sub-menu items. It supports two interfaces: IActionComponent and IPMPersist. We provide the implementation for IActionComponent in XMLMrkAction.cpp. The other default implementation is provided by the API.
