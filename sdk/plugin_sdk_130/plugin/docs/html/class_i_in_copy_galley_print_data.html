Detailed Description

Interface that contains data items used to configure InCopy to print documents in galley mode. The interface is persist to session workspace.

InCopy using IInCopyGalleyPrintData to configure the print dialog, then copy the data into corresponding IPrintData data field, for the print command using IPrintData.
