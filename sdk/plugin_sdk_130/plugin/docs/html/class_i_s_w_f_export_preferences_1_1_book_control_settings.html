Detailed Description

Book control (page curl) settings are used to determine how the page curl behavior works in SWF files exported from InDesign.

fUseBookControl may be used to enable or disable the page curl behavior.

fShowPageSlopeAtRest may be used to control whether the 'page spine' gradient will be shown even when no page curl transition is in effect. By default, the gradient is only shown while a page curl transition is in progress or while a user is performing a manual page turn by clicking a corner and dragging.

fShowCornerTease may be used to control whether a 'tease' animation is used when a user mouses over the corners of the page. The tease is meant to highlight that a given corner is interactive.
