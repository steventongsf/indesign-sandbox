Detailed Description

There is already well implemented scripting DOM in InDesign server for Middle Eastern (ME) text. BscMEScriptProvider provides the scripting method to access the ME features. It illustrates how to handle script elements such as page and paragraph, apply text attributes for ME text, apply world-ready composer on text.
