Detailed Description

 Represents a page in a spread (kSpreadBoss) or master spread (kMasterPagesBoss).

A page is responsible for drawing its outline, margins (IMargins), column guides (IColumns) and other aspects of a page's appearance. The association between a page (kPageBoss) and its master page is implemented by interface IMasterPage. A page is owned by a spread-layer (kSpreadLayerBoss) called the pages layer that is in turn owned by a spread; there are methods on ISpread to obtain the pages layer. Pages themselves do not own content, they represent the visual bounds within the spread that is considered a page. Pages are cached in the page-list (IPageList) on kDocBoss, this list is very handy if you want to quickly access the pages in a document.
