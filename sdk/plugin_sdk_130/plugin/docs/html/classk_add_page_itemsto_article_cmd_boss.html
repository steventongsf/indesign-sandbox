Detailed Description

 Wrapper command that can be scheduled by the ProcessDragDropCommand(). During drag drop operation we

shouldn't put up any UI. However, if we require to do so then we need to do so vai scheduling command. #2762748 This command adds dragged objects to the specified article & shows warning dialog if user tries to add already added objects.
