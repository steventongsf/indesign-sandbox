Detailed Description

 Duplicates Object style groups (kStyleGroupBoss).

The duplicated style groups are located as peers of the style group being duplicated.

Item list input
References the style groups (kStyleGroupBoss) being duplicated.
Item list output
None
Regular observer notification
  • theChange: kDuplicateObjectStyleGroupCmdBoss
  • theSubject: kWorkspaceBoss, kDocWorkspaceBoss
  • protocol: IID_ISTYLEGROUPINFO
  • changedBy: command instance
Lazy observer notification
  • theSubject: kWorkspaceBoss, kDocWorkspaceBoss
  • protocol: IID_ISTYLEGROUPINFO
  • data: ListLazyNotificationData<UID> referencing the style groups(kStyleGroupBoss) added by the command
